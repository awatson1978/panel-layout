## panel-layout

https://travis-ci.org/awatson1978/panel-layout.svg?branch=master

####  Version  
Meteor 1.0.3.1

````sh
# Should be as simple as cloning the repository...  
git clone https://github.com/awatson1978/panel-layout.git

# And then running it...
meteor
````
